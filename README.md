# About

backend project for create tasks together with someone, where you can create a collection, add people with different permissions to it, and track tasks through their statuses.

# Main Stack

- Nodejs
- TypeScript
- Fastify
- Postgres
- Docker
- jest + supertest

# How to run it

1. Clone this repository

   ```
   git clone https://github.com/soirrex/joint-tasks.git
   ```

2. Run in docker compose

   ```
   docker compose up
   ```

3. open documentation: `http://localhost:3000/docs`
